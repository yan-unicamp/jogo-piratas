Para adicionar as músicas com a temática de One Piece, coloque os arquivos `.mp3` ou `.ogg` nesta pasta com os seguintes nomes exatamente:

- menu.mp3 (Tocará na tela inicial)
- mapa.mp3 (Tocará na tela do mapa)
- batalha.mp3 (Tocará durante os combates)
- loja.mp3 (Tocará nas ilhas de loja)
- descanso.mp3 (Tocará nas ilhas de fogueira/descanso)

Nota: Se os arquivos não estiverem aqui, o jogo ainda rodará normalmente sem travar (apenas ficará mudo).
Você pode usar tanto arquivos com a extensão .mp3 quanto .ogg, basta garantir que o nome esteja correto. Se usar .ogg, avise para atualizarmos o gerenciador de áudio, mas por padrão ele está procurando por .mp3!
